How is this built?

  • On Xiami Ultra laptop, running Linux Mint 22. Default dual-boot installation with Windows 11 that the laptop shipped with works flawlessly, once Win11 disk encryption is turned off.
  • Content is written using markdown.
  • Often typed using AmazonBasics Pro Mechanical keyboard
  • Edited using neovim, with LazyVim (config) on WezTerm (config).
  • Static site is generated using Zola with tabi theme.
  • Objective is to have #PureContent => No Javascript / Cookies / Tracking. Current theme has cookies and javascript though.
